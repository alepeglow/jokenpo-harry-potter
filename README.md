# JokenPô

## Descrição
Este projeto é um jogo de **JokenPô** desenvolvido com **HTML, CSS e JavaScript**.  
O usuário pode escolher entre **pedra, papel ou tesoura**, enquanto o computador faz uma jogada aleatória.  
Ao final de cada rodada, o sistema mostra o resultado e atualiza o placar na tela.

## Funcionalidades
- Escolha entre pedra, papel ou tesoura
- Jogada aleatória da máquina
- Exibição do resultado da rodada
- Atualização dinâmica do placar do jogador e da máquina
- Interface visual temática

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Estrutura do projeto
```bash
.

```

## Como executar o projeto
1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` no navegador.

## Como jogar
1. Clique em uma das opções disponíveis:
   - Pedra
   - Papel
   - Tesoura
2. O computador fará uma escolha aleatória.
3. O resultado será exibido na tela.
4. O placar será atualizado automaticamente.

## Lógica do projeto
O JavaScript do projeto:
- captura a jogada do usuário
- gera uma jogada aleatória para o computador
- compara as duas escolhas
- define o vencedor da rodada
- atualiza o texto de resultado
- incrementa o placar do jogador ou da máquina
